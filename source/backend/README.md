# EDD Backend

## Installation
To install the required packages, execute the following command:
```bash
pip3 install -r requirements.txt
```

## Run
Execute the following command to start the backend server. The server will be listening on port 9000.
```bash
python3 main.py
```

To set a different port, use the following command:
```bash
python3 main.py --port 8080
```
